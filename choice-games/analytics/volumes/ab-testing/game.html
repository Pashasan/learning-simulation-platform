Chapter 1

Experiment Design

Why randomization is the foundation of causal inference

Dana Head of Product

"We redesigned the thumbnails for our top shows. But before rolling out to everyone, we need to test them. Some team members want to just show the new thumbnails to our most active users first. Others say we should randomize."

In an A/B test, we randomly assign users to see either the old design (control) or the new design (treatment):

🔎 Observational

Users self-select into groups. Active users may differ from inactive ones in ways that affect clicks.

🎲 Randomized

Random assignment balances groups on all factors — known and unknown.

Chapter 1

Randomization vs Observation

📚 The Setup

StreamVerse wants to test new thumbnails. One proposal: show them only to users who logged in this week. The other: randomly assign all users.

Why is random assignment better than showing new thumbnails only to active users?
It balances all factors — measured and unmeasured — across groups
Active users are more representative of the overall user base, giving cleaner data for analysis
Both approaches produce the same result
It requires fewer users to reach significance

What Randomization Guarantees

📊 Balance Check Results

Treatment: avg age = 32.1, 64% mobile, avg sub = 14 months

All differences tiny and statistically insignificant (p > 0.3).

These balance check results confirm that randomization...
Perfectly eliminates all possible bias
Created balanced groups on observed factors, giving confidence in balance on unobserved factors too
Was unnecessary since the groups look similar
Guarantees the new thumbnails will work
Chapter 2

Statistical Analysis

Testing whether the observed difference is real

Dana Head of Product

"The results are in! The new thumbnails got a 27.8% click rate vs 24.5% for the old ones. That's a 3.3 percentage point increase! But can we trust this?"

Before celebrating, we need to ask: Could this difference have occurred by chance? The chi-square test compares what we observed to what we'd expect if there were no difference.

Chapter 2

Interpreting p-values

📊 Chi-Square Test Results

Null hypothesis: New thumbnails have no effect on clicks.

Test statistic: χ² = 8.95

p-value = 0.003

What does a p-value of 0.003 mean in this context?
There is a 0.3% probability the new thumbnails are better
The click rate improved by exactly 0.3%
If thumbnails had no effect, there's only a 0.3% chance of seeing this large a difference
We can be 99.7% confident the new thumbnails are superior in every metric measured

Why Use Regression?

Maya Senior Data Scientist

"The chi-square says it's significant. But I also ran a regression controlling for device, subscription tier, and time of day. The treatment effect barely changed."

📊 Simple Comparison

Treatment effect: +3.3pp

Just comparing group means

📈 With Controls

Treatment effect: +3.1pp

Controlling for device, tier, time

The treatment effect barely changed when adding controls (+3.3pp → +3.1pp). This suggests:
Randomization successfully balanced confounders
The control variables are useless and should be dropped from the model entirely
The experiment failed because a good study should show very different results with and without controls
Only the regression is trustworthy, not the comparison
Chapter 3

Advanced Testing

Multiple comparisons and practical significance

Dana Head of Product

"The design team created 4 different thumbnail variants (A, B, C, D). We tested all of them against the control. Variant C has p = 0.04. Should we celebrate?"

When you test multiple variants, you increase the chance of finding a "significant" result by pure luck. This is the multiple testing problem.

Chapter 3

False Positive Explorer

The Multiple Testing Trap

Each test has a 5% false positive rate. But with multiple tests, the overall risk compounds rapidly.

4
simultaneous tests
18.5%
chance of ≥1 false positive
Bonferroni threshold:
0.0125
(0.05 ÷ number of tests)
With 4 tests, there's nearly a 1-in-5 chance of a false positive! The Bonferroni correction lowers the threshold to 0.05/k to keep the overall error rate at 5%.
Chapter 3

Multiple Testing Correction

📊 Results: 4 Variants vs Control

Variant A: p = 0.32

Variant B: p = 0.15

Variant C: p = 0.04 (significant at 0.05)

Variant D: p = 0.78

Bonferroni threshold: 0.05 / 4 = 0.0125

After Bonferroni correction for 4 comparisons, is Variant C (p = 0.04) still significant?
Yes — p = 0.04 is below 0.05, so it remains significant
No — the corrected threshold is 0.0125, and 0.04 exceeds it
Yes — Bonferroni is only needed when you have more than 10 simultaneous comparisons to worry about
Just pick whichever variant has the lowest p-value
Chapter 3

Statistical vs Practical Significance

Raj Data Engineer

"I ran the test on our full 2 million user sample. Click rate went from 24.50% to 24.62%. Highly significant: p = 0.001! But that's only +0.12pp. Worth the deploy effort?"

📈 Statistically Significant

p = 0.001 — the effect is real.

💰 Practically Significant?

+0.12pp ≈ 2,400 extra clicks/day. Worth the engineering work?

When can a result be statistically significant but not practically significant?
When the experiment ran longer than planned
When the sample is very large but the effect is tiny
When the p-value falls exactly on the 0.05 boundary
This is impossible — statistical significance always means the effect matters for business decisions

A/B Testing Cheat Sheet — Save This!

Randomization
Random assignment balances all factors across groups
The foundation of causal inference
Confounding
A variable that affects both treatment and outcome
Randomization eliminates confounding
Chi-Square Test
Tests independence of two categorical variables
χ² = ∑(O − E)² / E
P-value
P(data | null hypothesis) — not P(hypothesis | data)
p < 0.05 is the conventional threshold
Bonferroni Correction
Adjusts threshold for multiple comparisons
New threshold = α / number of tests
Effect Size
Magnitude of the treatment effect (e.g., +3.3pp)
Always report alongside p-values
Practical Significance
Whether the effect is large enough to matter
Statistical ≠ Practical significance
Segmentation
Checking if the effect varies across groups
Interaction terms in regression
